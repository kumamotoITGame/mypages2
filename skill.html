開発経験・強み

  • Unity Editor拡張を用いた開発支援ツールを設計・実装。 アセットインポート時にオブジェクトの自動分類、 階層構築、必要コンポーネントの自動アタッチ、 Prefab生成までを自動化し、 制作フローの効率化と人的ミス削減を実現。

  • Unity ShaderLabを用いたシェーダー制作・カスタマイズ経験。 描画表現の調整やマテリアル制御など、 ビジュアル面の実装にも対応可能。

  • Unreal Engineでは、 BlueprintおよびC++を用いたゲーム機能実装に加え、 不具合調査・修正、既存システムの機能拡張、 保守対応を経験。
    既存コードを理解した上で、 安定性を維持しながら機能追加を行う対応力を保有。

  • 新規開発メンバー向けに、 C#製セットアップ支援ツールを開発。 開発環境構築の自動化により、 導入コスト削減とチーム参加までの時間短縮を実現。

  • 数学・物理演算分野にも関心があり、 破壊演出における物理シミュレーション、 広大な海の不安定な水面の波とそれに同期したオブジェクト配置を意識した大量オブジェクト処理時の最適化、 ゲームプレイ品質と処理負荷の両立を意識した実装を行う。

  • GitHub Desktopを利用した バージョン管理・チーム開発経験。 複数人開発におけるブランチ運用や アセット共有フローへの理解あり。

  • プログラミングに加え、 3Dモデリング、イラスト制作、音楽制作まで対応可能。 技術・デザイン両面からゲーム開発を行える マルチスキル型クリエイター。
